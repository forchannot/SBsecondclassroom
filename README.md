# 第二课堂网络学习刷分脚本
## 原理：
使用微信小程序要等待五分钟才能答题，为了突破此限制，直接向抓包获取的接口提交所有可能的答案组合(A,B,C,D,AB,AC...)  
* 需要输入学号，不需要输入密码
* 由于系统限制，每天只能刷两个思政学分，
* 可以用`gthub action`或者`云函数`每天定时执行
